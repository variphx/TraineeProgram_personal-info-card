# Personal Info Card

## Brief

This project contains personal informations of TraineeProgram2023 Team T1's members, displayed via website formatting.

## How to run

Install dependencies:

```Bash
npm install
```

View in developper mode:

```Bash
npm run dev
```

Build project:

```Bash
npm run build
```

Preview built project:

```Bash
npm run preview
```
